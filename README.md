# 簡介
在無x window環境下使用virtualbox,可以節省主機的空間和資源, 雖然virtualbox有提供命令列指令, 但
使用時難免要查找相對應指令, 太費時了, 所以寫一個簡單的CUI介面來操作, 就會比較便利許多。
